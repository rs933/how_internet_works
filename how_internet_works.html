How Does
Internet Work?

The Internet (or internet) is the global system of interconnected computer networks that
uses the Internet protocol suite (TCP/IP) to communicate between networks and devices.

What Is Domain
In Internet?

A domain name is an identification string that defines a realm of administrative
autonomy, authority or control within the Internet.

What Is Internet?


Formal Definition

"The Internet, a loosely-organized international collaboration of autonomous, interconnected networks, supports host-to-host communication through voluntary adherence to open protocols and procedures defined by Internet Standards [1]."

My Definition

A collection of multiple different pieces of computer and non-computer equipment talking to one another by means of a communications network either wired or wireless.

How An Internet Message Is Sent?

  • Data being sent is divided into packets (Max size = 64K, 65535 bytes ).
  • Packet has a header with some technical information and a FROM address, TO address and the actual data. Each header is 60 bytes.
  • Packets divided in to multiple parts contain the order of the packet.
  • Packets move on the Internet along different paths each taking different times.
  • Packets are reordered
  • Received packets are acknowledged back to the sender.
  • Sender resends packets that do not acknowledge.

Command To See Internet Working

Command Meaning
IPConfig /all Your local connection /all shows MAC address, etc.

Netstat -an See open ports used by your machine.
